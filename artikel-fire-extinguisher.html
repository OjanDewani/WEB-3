ABOUT PRODUCT

Jenis APAR & Fungsinya

Fire extinguisher atau APAR merupakan alat pemadam api ringan yang digunakan untuk memadamkan kebakaran pada tahap awal sebelum api berkembang lebih besar.

STARVVO menyediakan berbagai jenis APAR profesional dengan media pemadam yang berbeda sesuai dengan klasifikasi bahaya kebakaran. Setiap jenis APAR memiliki fungsi khusus untuk memberikan perlindungan maksimal pada berbagai area dan kebutuhan operasional.

01

Dry Chemical Powder

APAR Dry Chemical Powder merupakan jenis alat pemadam api yang paling umum digunakan untuk berbagai klasifikasi kebakaran.

Jenis APAR ini efektif digunakan untuk kebakaran kelas A, B, dan C seperti kebakaran benda padat, cairan mudah terbakar, maupun kebakaran akibat listrik.

✓ Multi Purpose Fire Extinguisher
✓ Cocok Untuk Area Industrial
✓ Efektif Untuk Kebakaran Listrik
02

Foam AFFF

APAR Foam AFFF digunakan untuk memadamkan kebakaran cairan mudah terbakar seperti bensin, solar, oli, dan bahan kimia tertentu.

Foam bekerja dengan membentuk lapisan busa pada permukaan cairan sehingga mampu menghentikan suplai oksigen dan mencegah api menyala kembali.

✓ Liquid Fire Protection
✓ Mencegah Api Menyala Kembali
✓ Cocok Untuk Area Fuel Storage
03

CO2 (Carbon Dioxide)

APAR CO2 digunakan untuk memadamkan kebakaran akibat listrik dan perangkat elektronik tanpa meninggalkan residu.

Jenis APAR ini sangat cocok digunakan pada ruang server, panel listrik, laboratorium, dan area dengan perangkat sensitif.

✓ Clean Agent Fire Extinguisher
✓ Tidak Meninggalkan Residu
✓ Aman Untuk Perangkat Elektronik
04

AF-11 Clean Agent

AF-11 Clean Agent merupakan media pemadam modern yang aman digunakan untuk area elektronik dan perangkat sensitif.

Media ini bekerja cepat tanpa merusak perangkat dan tidak meninggalkan residu setelah proses pemadaman.

✓ Modern Clean Agent
✓ Fast Fire Suppression
✓ Aman Untuk Data Center
05

AF-36 Clean Agent

AF-36 Clean Agent merupakan solusi fire suppression modern dengan kemampuan pemadaman cepat untuk berbagai area berisiko tinggi.

Sistem ini cocok digunakan pada ruang server, control room, ruang listrik, dan fasilitas industrial modern.

✓ Professional Fire Protection
✓ Industrial Fire Safety
✓ High Performance Clean Agent
06

F-500 Encapsulator Agent

F-500 Encapsulator Agent merupakan teknologi pemadam kebakaran modern yang menggunakan mekanisme encapsulation untuk membantu mengendalikan panas, mengisolasi bahan bakar, dan mempercepat proses pemadaman api secara efektif.

Media pemadam ini sangat efektif digunakan pada kebakaran kelas A, kelas B, baterai lithium-ion, kendaraan listrik, area industri, gudang logistik, fasilitas manufaktur, serta lingkungan dengan risiko kebakaran berenergi tinggi.

Dibandingkan media pemadam konvensional, F-500 mampu menurunkan suhu dengan cepat, mengurangi kemungkinan re-ignition (api menyala kembali), serta membantu meningkatkan keselamatan personel dan aset perusahaan.

✓ Advanced Encapsulator Technology
✓ Effective For Lithium Battery Fires
✓ Rapid Cooling Performance
✓ Reduces Re-Ignition Risk
