Product Description

FM200 Fire Suppression System STARVVO merupakan solusi proteksi kebakaran modern yang bekerja menggunakan clean agent HFC-227ea. Sistem ini dirancang untuk memadamkan api secara cepat pada tahap awal tanpa menyebabkan kerusakan terhadap peralatan elektronik, dokumen penting, maupun aset bernilai tinggi.

Ketika kebakaran terdeteksi oleh fire alarm system, panel kontrol akan mengaktifkan pelepasan agent FM200 ke area yang dilindungi. Gas akan menyebar secara merata dan menghentikan proses pembakaran dalam hitungan detik sehingga api dapat dipadamkan sebelum berkembang menjadi kebakaran besar.
