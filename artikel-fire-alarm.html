ABOUT SYSTEM

Sistem Deteksi Kebakaran Modern

Fire alarm system merupakan sistem proteksi kebakaran yang berfungsi untuk mendeteksi indikasi kebakaran dan memberikan peringatan secara cepat kepada penghuni bangunan.

STARVVO menyediakan berbagai perangkat fire alarm system profesional untuk kebutuhan industrial dan commercial building dengan teknologi modern yang membantu meningkatkan keselamatan bangunan dan operasional perusahaan.

01

Smoke Detector

Smoke detector merupakan perangkat fire alarm yang digunakan untuk mendeteksi asap sebagai indikasi awal kebakaran.

Sistem ini bekerja secara otomatis dan membantu memberikan peringatan dini sebelum api berkembang lebih besar.

✓ Early Smoke Detection
✓ Automatic Fire Detection
✓ Building Safety Protection
02

Heat Detector

Heat detector digunakan untuk mendeteksi kenaikan suhu abnormal pada area tertentu yang berpotensi mengalami kebakaran.

Jenis detector ini cocok digunakan pada area yang memiliki potensi asap tinggi seperti kitchen, workshop, dan industrial area.

✓ High Temperature Detection
✓ Industrial Fire Safety
✓ Fast Warning System
03

Panel MCFA

MCFA Panel merupakan pusat kontrol utama pada sistem fire alarm yang berfungsi menerima sinyal dari detector dan mengaktifkan alarm system.

Panel ini membantu monitoring sistem proteksi kebakaran secara terintegrasi pada seluruh area bangunan.

✓ Main Fire Alarm Control
✓ Integrated Monitoring System
✓ Building Fire Protection
04

Manual Call Point

Manual Call Point digunakan untuk mengaktifkan alarm kebakaran secara manual ketika terjadi kondisi darurat.

Perangkat ini biasanya dipasang pada jalur evakuasi, area publik, dan titik strategis dalam bangunan.

✓ Emergency Alarm Activation
✓ Easy To Operate
✓ Public Area Protection
05

Alarm Bell

Alarm bell berfungsi memberikan peringatan suara ketika sistem fire alarm mendeteksi kebakaran.

Sistem ini membantu proses evakuasi penghuni bangunan dengan memberikan sinyal bahaya secara cepat dan jelas.

✓ Emergency Warning System
✓ Fast Evacuation Alert
✓ Building Alarm Protection
06

Horn Strobe

Horn strobe merupakan perangkat fire alarm yang memberikan peringatan suara dan cahaya secara bersamaan ketika terjadi kebakaran.

Sistem ini membantu memberikan notifikasi visual dan audio untuk meningkatkan keselamatan penghuni bangunan.

✓ Audio Visual Warning
✓ Emergency Notification System
✓ Modern Fire Alarm Device
